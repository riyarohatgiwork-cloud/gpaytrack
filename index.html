How to get your Client ID
1
Go to console.cloud.google.com → Sign in → Click New Project → Name it GPay Track → Create
2
Go to APIs & Services → Library → Search Google Drive API → Enable it
3
Go to APIs & Services → OAuth consent screen → Choose External → Fill your app name & email → Add your email as Test User → Save
4
Go to Credentials → Create Credentials → OAuth Client ID → Choose Web application → Under Authorised JavaScript origins add your app URL → Create
5
Copy the Client ID shown and paste it above
